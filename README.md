# Openenv
Smart Support Routing AI (OpenEnv) is an intelligent system designed to automatically analyze and classify customer support tickets using an AI-based approach. The application processes incoming queries, predicts the appropriate department such as technical support, billing, or general inquiries, and assigns a priority level based on the urgency of the issue. It operates within a reward-based environment where the system evaluates the correctness of predictions and provides a score, helping improve decision-making over time. Built using Python and integrated with a user-friendly Gradio interface, the project allows users to interactively test the model by selecting different difficulty levels and viewing real-time results. With Docker support for seamless deployment and a modular architecture for easy scalability, this project serves as a strong foundation for building advanced AI-driven customer support solutions and is suitable for academic projects, hackathons, and real-world applications.
